<Info>
コンテキストが重要です。変数が返す `N/A` 現在のコマンドまたはイベントがソース値を提供しない場合。追加 `|fallback text` よりフレンドリーなフォールバックが必要な場合は、右中括弧の前に置きます。
</Info>


| 変数 | 返されるもの | 出力例 | で利用可能 |
| --- | --- | --- | --- |
| <code>&#123;ticket.channel&#125;</code> | チャンネルの言及またはチャンネルの値。出典：チケット。 | <code>チケットチャネルの例</code> | チケットパネル、チケットログ、チケットメッセージ |
| <code>&#123;ticket.channel_id&#125;</code> | Discord チャンネルID。出典：チケット。 | <code>チケットチャネルIDの例</code> | チケットパネル、チケットログ、チケットメッセージ |
| <code>&#123;ticket.claimed_by&#125;</code> | チケットの請求者。 | <code>請求されたチケットの例</code> | チケットパネル、チケットログ、チケットメッセージ |
| <code>&#123;ticket.claimed_by_id&#125;</code> | チケットの ID によって要求されました。 | <code>ID によって要求されたチケットの例</code> | チケットパネル、チケットログ、チケットメッセージ |
| <code>&#123;ticket.claimed_by_mention&#125;</code> | チケットの言及による請求。 | <code>言及によって請求されたチケットの例</code> | チケットパネル、チケットログ、チケットメッセージ |
| <code>&#123;ticket.closed_at&#125;</code> | チケットの受付時間は です。 | <code>終了したチケットの例</code> | チケットパネル、チケットログ、チケットメッセージ |
| <code>&#123;ticket.closed_at_timestamp&#125;</code> | 閉店時刻の Unix タイムスタンプ。 | <code>1781460000</code> | チケットパネル、チケットログ、チケットメッセージ |
| <code>&#123;ticket.closed_by&#125;</code> | チケットの締切日。 | <code>クローズされたチケットの例</code> | チケットパネル、チケットログ、チケットメッセージ |
| <code>&#123;ticket.closed_by_id&#125;</code> | チケットの ID によってクローズされました。 | <code>ID によってクローズされたチケットの例</code> | チケットパネル、チケットログ、チケットメッセージ |
| <code>&#123;ticket.closed_by_mention&#125;</code> | チケットの言及によりクローズ。 | <code>メンションによってクローズされたチケットの例</code> | チケットパネル、チケットログ、チケットメッセージ |
| <code>&#123;ticket.created_at&#125;</code> | Discord-フォーマットされた作成時間。出典：チケット。 | <code>&lt;t:1781460000:F&gt;</code> | チケットパネル、チケットログ、チケットメッセージ |
| <code>&#123;ticket.created_at_timestamp&#125;</code> | 作成時の Unix タイムスタンプ。出典：チケット。 | <code>&lt;t:1781460000:F&gt;</code> | チケットパネル、チケットログ、チケットメッセージ |
| <code>&#123;ticket.id&#125;</code> | Discord ID。出典：チケット。 | <code>123456789012345678</code> | チケットパネル、チケットログ、チケットメッセージ |
| <code>&#123;ticket.number&#125;</code> | チケットの番号。 | <code>チケット番号の例</code> | チケットパネル、チケットログ、チケットメッセージ |
| <code>&#123;ticket.reason&#125;</code> | 理由のテキスト。出典：チケット。 | <code>理由は示されていません。</code> | チケットパネル、チケットログ、チケットメッセージ |
| <code>&#123;ticket.status&#125;</code> | 現在のステータス。出典：チケット。 | <code>チケットステータスの例</code> | チケットパネル、チケットログ、チケットメッセージ |
| <code>&#123;ticket.type&#125;</code> | タイプまたはカテゴリ名。出典：チケット。 | <code>チケットタイプの例</code> | チケットパネル、チケットログ、チケットメッセージ |
| <code>&#123;ticket.user&#125;</code> | チケットのユーザー。 | <code>チケットユーザーの例</code> | チケットパネル、チケットログ、チケットメッセージ |
| <code>&#123;ticket.user_id&#125;</code> | チケットのユーザーID。 | <code>チケットユーザーIDの例</code> | チケットパネル、チケットログ、チケットメッセージ |
| <code>&#123;ticket.user_mention&#125;</code> | チケットのユーザーメンション。 | <code>チケットユーザーの言及の例</code> | チケットパネル、チケットログ、チケットメッセージ |
