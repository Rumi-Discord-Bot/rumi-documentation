---
title: "引数変数"
description: "Rumi がサポートする 16 個の引数変数。"
icon: "brackets-curly"
---

<Info>
コンテキストが重要です。変数が返す `N/A` 現在のコマンドまたはイベントがソース値を提供しない場合。追加 `|fallback text` よりフレンドリーなフォールバックが必要な場合は、右中括弧の前に置きます。
</Info>


| 変数 | 返されるもの | 出力例 | で利用可能 |
| --- | --- | --- | --- |
| <code>&#123;arg&#125;</code> | カスタム コマンドまたは一致した自動応答トリガーの後に入力されたすべて。 | <code>振りかける青いケーキ</code> | カスタムコマンド、自動応答 |
| <code>&#123;arg.1&#125;</code> | コマンドまたはトリガーの後の最初の引数。 | <code>青</code> | カスタムコマンド、自動応答 |
| <code>&#123;arg.2+&#125;</code> | 引数 2 とその後のすべて。 | <code>スプリンクルを添えたケーキ</code> | カスタムコマンド、自動応答 |
| <code>&#123;arg.count&#125;</code> | 解析されたコマンド引数の数。 | <code>カスタム コマンド ガイドの構文と例を参照してください。</code> | カスタムコマンド。カスタムデータヘルパーには自動応答の --db も必要です |
| <code>&#123;arg.raw&#125;</code> | 変更されていないコマンド引数のテキスト。 | <code>カスタム コマンド ガイドの構文と例を参照してください。</code> | カスタムコマンド。カスタムデータヘルパーには自動応答の --db も必要です |
| <code>&#123;args&#125;</code> | コマンドの後に指定されたすべてのテキスト。別名 &#123;引数&#125;. | <code>カスタム コマンド ガイドの構文と例を参照してください。</code> | カスタムコマンド。カスタムデータヘルパーには自動応答の --db も必要です |
| <code>&#123;args.count&#125;</code> | 解析されたコマンド引数の数。 | <code>カスタム コマンド ガイドの構文と例を参照してください。</code> | カスタムコマンド。カスタムデータヘルパーには自動応答の --db も必要です |
| <code>&#123;args.raw&#125;</code> | 変更されていないコマンド引数のテキスト。 | <code>カスタム コマンド ガイドの構文と例を参照してください。</code> | カスタムコマンド。カスタムデータヘルパーには自動応答の --db も必要です |
| <code>&#123;arg.1+&#125;</code> | 引数 1 とその後のすべての引数。 1 を開始位置に置き換えます。 | <code>カスタム コマンド ガイドの構文と例を参照してください。</code> | カスタムコマンド。カスタムデータヘルパーには自動応答の --db も必要です |
| <code>&#123;args.rest.1&#125;</code> | 引数 1 とその後のすべての引数。 1 を開始位置に置き換えます。 | <code>カスタム コマンド ガイドの構文と例を参照してください。</code> | カスタムコマンド。カスタムデータヘルパーには自動応答の --db も必要です |
| <code>&#123;variable&#124;fallback text&#125;</code> | 通常のコンテキスト変数が使用できない場合は、フォールバック テキストを返します。 | <code>カスタム コマンド ガイドの構文と例を参照してください。</code> | カスタムコマンド。カスタムデータヘルパーには自動応答の --db も必要です |
| <code>&#123;name&#125;</code> | 宣言された値 &lt;名前&gt; または [名前] カスタムコマンド引数。 | <code>カスタム コマンド ガイドの構文と例を参照してください。</code> | カスタムコマンド。カスタムデータヘルパーには自動応答の --db も必要です |
| <code>&#123;arg.name&#125;</code> | 宣言された名前付きカスタムコマンド引数の値。 | <code>カスタム コマンド ガイドの構文と例を参照してください。</code> | カスタムコマンド。カスタムデータヘルパーには自動応答の --db も必要です |
| <code>&#123;arg1&#125;</code> | 番号付き引数のコンパクトなエイリアス。 1 を位置に置き換えます。 | <code>カスタム コマンド ガイドの構文と例を参照してください。</code> | カスタムコマンド。カスタムデータヘルパーには自動応答の --db も必要です |
| <code>&#123;args.1&#125;</code> | 番号付きの位置の引数。 | <code>カスタム コマンド ガイドの構文と例を参照してください。</code> | カスタムコマンド。カスタムデータヘルパーには自動応答の --db も必要です |
| <code>&#123;1&#125;</code> | 番号付きの位置の引数。 1 を任意の正の位置に置き換えます。 | <code>カスタム コマンド ガイドの構文と例を参照してください。</code> | カスタムコマンド。カスタムデータヘルパーには自動応答の --db も必要です |
