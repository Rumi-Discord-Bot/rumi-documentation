---
title: "レベル変数"
description: "Rumi がサポートする 15 のレベル変数。"
icon: "brackets-curly"
---

<Info>
コンテキストが重要です。変数が返す `N/A` 現在のコマンドまたはイベントがソース値を提供しない場合。追加 `|fallback text` よりフレンドリーなフォールバックが必要な場合は、右中括弧の前に置きます。
</Info>


| 変数 | 返されるもの | 出力例 | で利用可能 |
| --- | --- | --- | --- |
| <code>&#123;level&#125;</code> | レベルアップしたメンバーのレベルとXPデータ。 | <code>例レベル</code> | レベルメッセージ、レベルプレビュー |
| <code>&#123;level.new_rank&#125;</code> | メンバーが新たなレベルに到達しました。 | <code>12</code> | レベルメッセージ、レベルプレビュー |
| <code>&#123;level.next_rank&#125;</code> | 現在のレベルの次のレベル。 | <code>次のランクのレベルの例</code> | レベルメッセージ、レベルプレビュー |
| <code>&#123;level.old_rank&#125;</code> | レベルアップ前の以前のレベル。 | <code>11</code> | レベルメッセージ、レベルプレビュー |
| <code>&#123;level.progress&#125;</code> | 次のレベルに向かって進みます。 | <code>74%</code> | レベルメッセージ、レベルプレビュー |
| <code>&#123;level.rank_position&#125;</code> | メンバーのリーダーボードのランク位置。 | <code>レベルランク位置の例</code> | レベルメッセージ、レベルプレビュー |
| <code>&#123;level.server_xp&#125;</code> | レベルはサーバーXp。 | <code>サンプルレベルサーバーXP</code> | レベルメッセージ、レベルプレビュー |
| <code>&#123;level.server_xp_total&#125;</code> | サーバー XP レベルの合計。 | <code>例レベルのサーバー XP 合計</code> | レベルメッセージ、レベルプレビュー |
| <code>&#123;level.total_messages&#125;</code> | レベルの合計メッセージ数。 | <code>レベル合計メッセージの例</code> | レベルメッセージ、レベルプレビュー |
| <code>&#123;level.user_xp&#125;</code> | メンバーの現在の XP 値。 | <code>8,420</code> | レベルメッセージ、レベルプレビュー |
| <code>&#123;level.user_xp_total&#125;</code> | ユーザーXpのレベルの合計。 | <code>例レベルのユーザー XP 合計</code> | レベルメッセージ、レベルプレビュー |
| <code>&#123;level.voice_seconds&#125;</code> | レベルの音声秒数。 | <code>例レベルの音声秒数</code> | レベルメッセージ、レベルプレビュー |
| <code>&#123;level.voice_time&#125;</code> | レベルのボイスタイム。 | <code>レベルボイスタイムの例</code> | レベルメッセージ、レベルプレビュー |
| <code>&#123;level.xp_needed&#125;</code> | 次のレベルに必要なXP。 | <code>9,000</code> | レベルメッセージ、レベルプレビュー |
| <code>&#123;xp&#125;</code> | レベルアップしたメンバーのレベルとXPデータ。 | <code>例 xp</code> | レベルメッセージ、レベルプレビュー |
