---
title: "カスタムデータベース変数"
description: "Rumi でサポートされている 10 個のカスタムデータベース変数。"
icon: "brackets-curly"
---

<Info>
コンテキストが重要です。変数が返す `N/A` 現在のコマンドまたはイベントがソース値を提供しない場合。追加 `|fallback text` よりフレンドリーなフォールバックが必要な場合は、右中括弧の前に置きます。
</Info>


| 変数 | 返されるもの | 出力例 | で利用可能 |
| --- | --- | --- | --- |
| <code>&#123;customdb.&lt;operation&gt;:namespace:key[:value]&#125;</code> | すべてのデータベース操作の長い形式のエイリアス: get、exists、set、inc、increment、dec、decrement、delete、del。 | <code>カスタム コマンド ガイドの構文と例を参照してください。</code> | カスタムコマンド。カスタムデータヘルパーには自動応答の --db も必要です |
| <code>&#123;db.dec:namespace:key:amount&#125;</code> | カスタムデータ番号を減らします。 | <code>カスタム コマンド ガイドの構文と例を参照してください。</code> | カスタムコマンド。カスタムデータヘルパーには自動応答の --db も必要です |
| <code>&#123;db.decrement:namespace:key:amount&#125;</code> | db.dec の長い形式のエイリアス。 | <code>カスタム コマンド ガイドの構文と例を参照してください。</code> | カスタムコマンド。カスタムデータヘルパーには自動応答の --db も必要です |
| <code>&#123;db.del:namespace:key&#125;</code> | db.delete のエイリアス。 | <code>カスタム コマンド ガイドの構文と例を参照してください。</code> | カスタムコマンド。カスタムデータヘルパーには自動応答の --db も必要です |
| <code>&#123;db.delete:namespace:key&#125;</code> | カスタムデータキーを削除します。 | <code>カスタム コマンド ガイドの構文と例を参照してください。</code> | カスタムコマンド。カスタムデータヘルパーには自動応答の --db も必要です |
| <code>&#123;db.exists:namespace:key&#125;</code> | カスタムデータキーが存在する場合は true を返し、それ以外の場合は false を返します。 | <code>カスタム コマンド ガイドの構文と例を参照してください。</code> | カスタムコマンド。カスタムデータヘルパーには自動応答の --db も必要です |
| <code>&#123;db.get:namespace:key&#125;</code> | 保存されたカスタム データ値を読み取ります。 | <code>42</code> | カスタムコマンド。 --db を使用した自動応答 |
| <code>&#123;db.inc:namespace:key:1&#125;</code> | 保存されている数値をインクリメントします。生の結果を非表示にするには、独自の行に配置します。 | <code>43</code> | カスタムコマンド。 --db を使用した自動応答 |
| <code>&#123;db.increment:namespace:key:amount&#125;</code> | カスタムデータ番号を増やします。 db.incの別名。 | <code>カスタム コマンド ガイドの構文と例を参照してください。</code> | カスタムコマンド。カスタムデータヘルパーには自動応答の --db も必要です |
| <code>&#123;db.set:namespace:key:value&#125;</code> | カスタムデータ値を保存します。単独の行では、突然変異は目に見える出力を生成しません。 | <code>カスタム コマンド ガイドの構文と例を参照してください。</code> | カスタムコマンド。カスタムデータヘルパーには自動応答の --db も必要です |
