Rumi サポート:

- 英語 （`en`)
- ドイツ語 (`de`)
- フランス語 (`fr`)
- オランダ語 (`nl`)
- 日本語 （`ja`)
- 簡体字中国語 (`zh-CN`)

ドキュメント Web サイトには、英語、オランダ語、ドイツ語、
スペイン語、日本語、簡体字中国語、クロアチア語、フランス語。スペイン語と
クロアチア語のドキュメントが利用可能だからといって、ボットの応答が
これらの言語ではまだ利用できません。

## 効果的な言語を表示する

```text
-language
```

応答には、現在の言語とその設定の出所が示されます。

## 個人の言語を設定する

```text
-language user de
```

個人の言語設定はあなたに適用されます。ダイレクトメッセージでは、 Rumi これを保管します
ユーザーレコードの設定を変更します。

## サーバー言語を設定します

```text
-language server de
```

サーバー全体の言語を変更するには、管理者の許可が必要です。の
設定はそのサーバーに保存されます。

<Info>
個人言語はサーバー言語より優先されます。まだの場合
個人的な言語を選択し、 Rumi サーバー設定を使用します。 Discordのロケール、
それから英語。
</Info>

確認は、新しく選択した言語ですぐに表示されます。
