<Info>
Der Kontext ist wichtig. Eine Variable gibt zurück `N/A` wenn der aktuelle Befehl oder das aktuelle Ereignis seinen Quellwert nicht bereitstellt. Hinzufügen `|fallback text` vor der schließenden Klammer, wenn Sie einen freundlicheren Fallback benötigen.
</Info>


| Variable | Was es zurückgibt | Beispielausgabe | Verfügbar in |
| --- | --- | --- | --- |
| <code>&#123;boost.count&#125;</code> | Zählwert. Quelle: Boost. | <code>128</code> | Nachrichten verstärken, Nachrichten aufrufen |
| <code>&#123;boost.next_tier&#125;</code> | Nächste Stufe für Boost. | <code>Beispiel für einen Boost auf die nächste Stufe</code> | Nachrichten verstärken, Nachrichten aufrufen |
| <code>&#123;boost.perks&#125;</code> | Vorteile für Boost. | <code>Beispiel für Boost-Vorteile</code> | Nachrichten verstärken, Nachrichten aufrufen |
| <code>&#123;boost.tier&#125;</code> | Stufe für Boost. | <code>Beispiel einer Boost-Stufe</code> | Nachrichten verstärken, Nachrichten aufrufen |
| <code>&#123;boost.user_since&#125;</code> | Benutzer seit für Boost. | <code>Beispiel Boost-Benutzer seit</code> | Nachrichten verstärken, Nachrichten aufrufen |
| <code>&#123;boost.user_since_timestamp&#125;</code> | Unix-Zeitstempel für Benutzer seit. | <code>1781460000</code> | Nachrichten verstärken, Nachrichten aufrufen |
