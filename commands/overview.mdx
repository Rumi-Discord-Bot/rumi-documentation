Rumi's in-Discord help menu is the source of truth for the commands available
to you:

```text
-help
```

Use `-help <command>` to inspect a specific command. For example:

```text
-help customcommand
```

Help entries can include:

- Command and subcommand names
- Descriptions and prefix aliases
- Usage and argument information
- Required user and bot permissions
- Server-only and NSFW restrictions
- Slash-command support

<Info>
This site explains workflows in depth. If a guide and Rumi's current help entry
differ, follow the bot's help entry.
</Info>

## Command aliases

Some commands have shorter aliases. For example, `cc` opens `customcommand`,
`rr` opens `reactionrole`, and `rank` opens `level`. Aliases run the same command
and use the same permissions and saved settings.

The help entry shows supported aliases. Do not assume every shortened word is
an alias; if it is not listed, use the full command name.

## Prefix-only commands

Rumi has more prefix commands than slash commands. This is intentional while
the slash catalog is being cleaned up. A command marked **Prefix only** must be
run with the server prefix even if similarly named slash commands appear in
Discord.

See [how to read command syntax](/commands/reading-syntax) for required values,
optional arguments, choices, flags, quotes, and subcommands.
