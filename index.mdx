## Command names stay consistent

Rumi localizes descriptions, replies, buttons, menus, and other bot-authored
text. Command names remain unchanged so guides and commands are consistent in
every language.

<Info>
Rumi itself is the source of truth for available commands. Run `-help`, then
open a command's help entry to see its current syntax and requirements.
</Info>
