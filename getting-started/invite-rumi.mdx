You need permission to add applications to the destination Discord server.

## Add the bot

1. Open [rumi.rocks](https://rumi.rocks).
2. Select **Add to Discord**.
3. Choose the server.
4. Review the requested permissions and authorize the installation.
5. Return to Discord and confirm that Rumi appears in the member list.

<Warning>
Only authorize Rumi through links published on `rumi.rocks` or the official
support server.
</Warning>

## Confirm access

Run:

```text
-help
```

You can also open Discord's slash-command picker and search for Rumi.

If Rumi cannot respond, confirm that it can view the channel, send messages,
embed links, and use application commands.
