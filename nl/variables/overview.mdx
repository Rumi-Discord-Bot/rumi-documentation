Deze referentie bevat **388 variabelen en helpers**. Het wordt gegenereerd vanuit hetzelfde register dat door de bot wordt gebruikt `-variables` commando, zodat de documentatie en runtime op één lijn blijven.

Gebruik een fallback wanneer een waarde mogelijk niet bestaat in de huidige context:

```text
{target.display_name|No target selected}
```
