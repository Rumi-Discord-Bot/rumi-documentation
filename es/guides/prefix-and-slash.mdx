## Comandos de prefijo

Los comandos de prefijo se escriben normalmente Discord mensaje. El prefijo predeterminado es
`-`:

```text
-command argument
```

La configuración del servidor puede proporcionar un prefijo personalizado. Cuando no esté seguro, mencione Rumi o
usar DiscordSelector de comandos de barra diagonal para descubrir los comandos disponibles.
