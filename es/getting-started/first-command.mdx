Rumi admite comandos de prefijo y comandos de barra diagonal seleccionados.

## Comando de prefijo

El prefijo predeterminado es un guión (`-`):

```text
-help
```

No agregue un espacio entre el prefijo y el nombre del comando a menos que el comando
los argumentos requieren uno.

## comando de barra diagonal

Tipo `/` en Discord, seleccionar Rumiy elija un comando del selector. Discord
muestra los argumentos disponibles y si cada uno de ellos es necesario.

<Tip>
Si un comando no aparece en el selector de barra diagonal, utilice su versión de prefijo y
controlar https://rumi.rocks/commands para ver si se admite la ejecución de barra diagonal.
</Tip>

## Buscar la sintaxis del comando

Usar `-help` para buscar comandos, luego ejecutar `-help <command>` para la corriente
descripción, sintaxis, alias y requisitos.
