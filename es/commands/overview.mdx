Rumiestá en-Discord El menú de ayuda es la fuente de verdad para los comandos disponibles.
A usted:

```text
-help
```

Usar `-help <command>` para inspeccionar un comando específico. Por ejemplo:

```text
-help customcommand
```

Las entradas de ayuda pueden incluir:

- Nombres de comandos y subcomandos
- Descripciones y alias de prefijos.
- Información de uso y argumentos.
- Permisos de usuario y bot requeridos
- Restricciones de solo servidor y NSFW
- Soporte de comando de barra diagonal

<Info>
Este sitio explica los flujos de trabajo en profundidad. Si una guía y Rumientrada de ayuda actual
difieren, siga la entrada de ayuda del bot.
</Info>

## Alias ​​de comando

Algunos comandos tienen alias más cortos. Por ejemplo, `cc` abre `customcommand`,
`rr` abre `reactionrole`, y `rank` abre `level`. Los alias ejecutan el mismo comando.
y usar los mismos permisos y configuraciones guardadas.

La entrada de ayuda muestra los alias admitidos. No asuma que cada palabra abreviada es
un alias; si no aparece en la lista, utilice el nombre completo del comando.

## Comandos de solo prefijo

Rumi tiene más comandos de prefijo que comandos de barra diagonal. Esto es intencional mientras
El catálogo de barras se está limpiando. Se debe utilizar un comando marcado **Sólo prefijo**.
ejecutar con el prefijo del servidor incluso si aparecen comandos de barra con nombres similares en
Discord.

argumentos, opciones, indicadores, comillas y subcomandos opcionales.
