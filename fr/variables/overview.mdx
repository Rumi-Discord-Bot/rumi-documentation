Cette référence contient **388 variables et assistants**. Il est généré à partir du même registre utilisé par le bot `-variables` commande, afin que la documentation et le runtime restent alignés.

Utilisez une solution de secours lorsqu'une valeur peut ne pas exister dans le contexte actuel :

```text
{target.display_name|No target selected}
```
