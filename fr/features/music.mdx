Courir `-help music` pour trouver les commandes musicales actuellement disponibles et leurs
syntaxe.

## Flux de base

1. Rejoignez un Discord canal vocal.
2. Exécutez la commande play avec un terme de recherche ou une URL pris en charge.
3. Utilisez les contrôles de file d'attente pour gérer la lecture.
4. Déconnecter Rumi lorsque la lecture est terminée.

<Note>
La disponibilité de la musique dépend du service de musique légère distinct. Si la musique
les commandes répondent mais la lecture ne peut pas démarrer, vérifiez l'état du service ou contactez
soutien.
</Note>

Utiliser `-help <command>` avant d'exécuter une commande musicale pour voir ses options actuelles
et les exigences.

## Contrôles communs

```text
-play <song, playlist, or supported URL>
-queue
-nowplaying
-pause
-resume
-skip
-volume 65
-seek 1:30
-loop track
-shuffle
```

Rumila file d'attente vocale de et les informations personnelles d'un utilisateur Spotify l'appareil est une lecture séparée
cibles. Des commandes telles que `-pause` contrôle Rumi dans Discord voix. Des commandes telles
comme `-spotify pause` contrôler les actifs de l'utilisateur lié Spotify Connectez l'appareil.

Voir [Spotify liaison et contrôles](./spotify) pour les playlists, j'ai aimé
chansons, appareils et mouvements Spotify pistes de la bibliothèque dans le Discord file d'attente vocale.
