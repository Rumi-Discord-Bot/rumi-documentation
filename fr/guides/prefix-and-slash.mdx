## Commandes de préfixe

Les commandes de préfixe sont saisies normalement Discord message. Le préfixe par défaut est
`-`:

```text
-command argument
```

La configuration du serveur peut fournir un préfixe personnalisé. En cas de doute, mentionnez Rumi ou
utiliser DiscordLe sélecteur de commandes slash de pour découvrir les commandes disponibles.

## Commandes barre oblique

Utilisation des commandes Slash DiscordL'interface de commande d'application de :

```text
/command
```

Discord valide les options de barre oblique avant d'envoyer la commande à Rumi. Quelques
les commandes restent uniquement avec préfixe lorsque leur format d'argument ou leur flux d'interaction le fait
ne correspond pas proprement à DiscordLe schéma de commande slash de .

## Localisation

Les noms des commandes ne changent pas avec la langue sélectionnée. Descriptions, réponses,
les messages de validation, les boutons, les menus et les étiquettes modales peuvent être localisés.
