Rumi prend en charge les commandes de préfixe et les commandes de barre oblique sélectionnées.

## Commande de préfixe

Le préfixe par défaut est un trait d'union (`-`):

```text
-help
```

N'ajoutez pas d'espace entre le préfixe et le nom de la commande, sauf si le nom de la commande
les arguments en nécessitent un.

## Commande barre oblique

Taper `/` dans Discord, sélectionner Rumi, et choisissez une commande dans le sélecteur. Discord
montre les arguments disponibles et si chacun est requis.

<Tip>
Si une commande n'apparaît pas dans le sélecteur de barre oblique, utilisez sa version de préfixe et
vérifier https://rumi.rocks/commands pour voir si l'exécution de barre oblique est prise en charge.
</Tip>

## Rechercher la syntaxe de la commande

Utiliser `-help` pour parcourir les commandes, puis exécutez `-help <command>` pour le courant
description, syntaxe, alias et exigences.
