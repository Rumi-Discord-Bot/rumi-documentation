## Les noms des commandes restent cohérents

Rumi localise les descriptions, les réponses, les boutons, les menus et autres éléments créés par des robots
texte. Les noms des commandes restent inchangés afin que les guides et les commandes soient cohérents dans
chaque langue.

<Info>
Rumi lui-même est la source de vérité pour les commandes disponibles. Courir `-help`, alors
ouvrez l'entrée d'aide d'une commande pour voir sa syntaxe et ses exigences actuelles.
</Info>
