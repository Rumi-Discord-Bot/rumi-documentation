<Info>
背景很重要。变量返回 `N/A` 当当前命令或事件未提供其源值时。添加 `|fallback text` 当您需要更友好的后备时，在右大括号之前。
</Info>


| 多变的 | 它返回什么 | 输出示例 | 可用于 |
| --- | --- | --- | --- |
| <code>&#123;if:&lt;left&gt;&lt;operator&gt;&lt;right&gt;&#125;yes&#123;else&#125;no&#123;endif&#125;</code> | 使用 ==、!= 比较值 &gt;, &lt;, &gt;=，或 &lt;=. | <code>请参阅自定义命令指南中的语法和示例。</code> | 自定义命令；自定义数据助手还需要自动回复器中的 --db |
| <code>&#123;if:&lt;value&gt; contains &lt;text&gt;&#125;yes&#123;else&#125;no&#123;endif&#125;</code> | 检查文本是否包含另一个值，忽略大小写。 | <code>请参阅自定义命令指南中的语法和示例。</code> | 自定义命令；自定义数据助手还需要自动回复器中的 --db |
| <code>&#123;if:exists &lt;value&gt;&#125;yes&#123;else&#125;no&#123;endif&#125;</code> | 检查解析值是否存在且不为 false。 | <code>请参阅自定义命令指南中的语法和示例。</code> | 自定义命令；自定义数据助手还需要自动回复器中的 --db |
| <code>&#123;if user has permission &lt;permission&gt;&#125;yes&#123;else&#125;no&#123;endif&#125;</code> | 检查一个 Discord 对用户或目标的权限。 | <code>请参阅自定义命令指南中的语法和示例。</code> | 自定义命令；自定义数据助手还需要自动回复器中的 --db |
| <code>&#123;if user has role @Role&#125;Allowed&#123;else&#125;Denied&#123;endif&#125;</code> | 检查成员是否具有角色。 | <code>允许</code> | 自定义命令、自动回复 |
| <code>&#123;if:&#123;user.id&#125;==&#123;target.id&#125;&#125;No self target.&#123;else&#125;OK&#123;endif&#125;</code> | 当安全条件通过或失败时显示不同的文本。 | <code>好的</code> | 自定义命令、自动回复 |
