<Info>
背景很重要。变量返回 `N/A` 当当前命令或事件未提供其源值时。添加 `|fallback text` 当您需要更友好的后备时，在右大括号之前。
</Info>


| 多变的 | 它返回什么 | 输出示例 | 可用于 |

| <code>&#123;ticket.claimed_by&#125;</code> | 门票领取者。 | <code>领取门票的示例</code> | 票务面板、票务日志、票务消息 |
| <code>&#123;ticket.claimed_by_id&#125;</code> | 通过 ID 领取门票。 | <code>通过 ID 领取的票证示例</code> | 票务面板、票务日志、票务消息 |
| <code>&#123;ticket.claimed_by_mention&#125;</code> | 通过提及领取门票。 | <code>通过提及索取的门票示例</code> | 票务面板、票务日志、票务消息 |

| <code>&#123;ticket.id&#125;</code> | Discord ID。来源：门票。 | <code>123456789012345678</code> | 票务面板、票务日志、票务消息 |
