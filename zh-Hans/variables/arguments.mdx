---
title: "参数变量"
description: "Rumi 支持 16 个参数变量。"
icon: "brackets-curly"
---

<Info>
背景很重要。变量返回 `N/A` 当当前命令或事件未提供其源值时。添加 `|fallback text` 当您需要更友好的后备时，在右大括号之前。
</Info>


| 多变的 | 它返回什么 | 输出示例 | 可用于 |
| --- | --- | --- | --- |
| <code>&#123;arg&#125;</code> | 在自定义命令或匹配的自动应答器触发器之后键入的所有内容。 | <code>蓝色蛋糕，撒上糖粉</code> | 自定义命令、自动回复 |
| <code>&#123;arg.1&#125;</code> | 命令或触发器之后的第一个参数。 | <code>蓝色的</code> | 自定义命令、自动回复 |
| <code>&#123;arg.2+&#125;</code> | 论点 2 及其之后的所有内容。 | <code>洒上蛋糕</code> | 自定义命令、自动回复 |
| <code>&#123;arg.count&#125;</code> | 已解析的命令参数的数量。 | <code>请参阅自定义命令指南中的语法和示例。</code> | 自定义命令；自定义数据助手还需要自动回复器中的 --db |
| <code>&#123;arg.raw&#125;</code> | 未修改的命令参数文本。 | <code>请参阅自定义命令指南中的语法和示例。</code> | 自定义命令；自定义数据助手还需要自动回复器中的 --db |
| <code>&#123;args&#125;</code> | 命令后提供的所有文本。别名 &#123;精氨酸&#125;. | <code>请参阅自定义命令指南中的语法和示例。</code> | 自定义命令；自定义数据助手还需要自动回复器中的 --db |
| <code>&#123;args.count&#125;</code> | 已解析的命令参数的数量。 | <code>请参阅自定义命令指南中的语法和示例。</code> | 自定义命令；自定义数据助手还需要自动回复器中的 --db |
| <code>&#123;args.raw&#125;</code> | 未修改的命令参数文本。 | <code>请参阅自定义命令指南中的语法和示例。</code> | 自定义命令；自定义数据助手还需要自动回复器中的 --db |
| <code>&#123;arg.1+&#125;</code> | 参数 1 及其后的每个参数。将 1 替换为起始位置。 | <code>请参阅自定义命令指南中的语法和示例。</code> | 自定义命令；自定义数据助手还需要自动回复器中的 --db |
| <code>&#123;args.rest.1&#125;</code> | 参数 1 及其后的每个参数。将 1 替换为起始位置。 | <code>请参阅自定义命令指南中的语法和示例。</code> | 自定义命令；自定义数据助手还需要自动回复器中的 --db |
| <code>&#123;variable&#124;fallback text&#125;</code> | 当正常上下文变量不可用时返回后备文本。 | <code>请参阅自定义命令指南中的语法和示例。</code> | 自定义命令；自定义数据助手还需要自动回复器中的 --db |
| <code>&#123;name&#125;</code> | 申报价值 &lt;姓名&gt; 或 [名称] 自定义命令参数。 | <code>请参阅自定义命令指南中的语法和示例。</code> | 自定义命令；自定义数据助手还需要自动回复器中的 --db |
| <code>&#123;arg.name&#125;</code> | 声明的命名自定义命令参数的值。 | <code>请参阅自定义命令指南中的语法和示例。</code> | 自定义命令；自定义数据助手还需要自动回复器中的 --db |
| <code>&#123;arg1&#125;</code> | 编号参数的紧凑别名。将 1 替换为位置。 | <code>请参阅自定义命令指南中的语法和示例。</code> | 自定义命令；自定义数据助手还需要自动回复器中的 --db |
| <code>&#123;args.1&#125;</code> | 编号位置的论证。 | <code>请参阅自定义命令指南中的语法和示例。</code> | 自定义命令；自定义数据助手还需要自动回复器中的 --db |
| <code>&#123;1&#125;</code> | 编号位置的论证。将 1 替换为任意正位置。 | <code>请参阅自定义命令指南中的语法和示例。</code> | 自定义命令；自定义数据助手还需要自动回复器中的 --db |
