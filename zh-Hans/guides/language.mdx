Rumi 支持：

- 英语 （`en`)
- 德语（`de`)
- 法语 （`fr`)
- 荷兰语（`nl`)
- 日本人 （`ja`)
- 简体中文（`zh-CN`)

该文档网站还提供英语、荷兰语、德语、
西班牙语、日语、简体中文、克罗地亚语和法语。西班牙语和
克罗地亚文档可用性并不意味着机器人响应
尚未提供这些语言版本。

## 查看有效语言

```text
-language
```

响应显示当前语言以及偏好的来源。

## 设置您的个人语言

```text
-language user de
```

个人语言设置适用于您。在私信中， Rumi 存储这个
您的用户记录中的偏好设置。

## 设置服务器语言

```text
-language server de
```

更改服务器范围的语言需要管理员权限。这
保存该服务器的首选项。

<Info>
您的个人语言优先于服务器语言。如果你还没有
选择个人语言， Rumi 使用服务器设置，然后 Discord的区域设置，
然后是英语。
</Info>

确认信息会立即以新选择的语言显示。
